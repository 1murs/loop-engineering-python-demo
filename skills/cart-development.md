# Cart Development Skill

Use this skill when working on cart calculation logic.

Rules:

1. Always read docs/SPEC.md first.
2. Do not guess business logic.
3. The source file is src/cart.py.
4. The test file is tests/test_cart.py.
5. Use pytest to verify the result.
6. The task is complete only when all tests pass.

Common mistakes:
- Forgetting quantity
- Applying discount before calculating the full total
- Returning a string instead of a number
